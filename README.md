# Steam Update Tracker - Issue Tracker

Welcome to the **Steam Update Tracker** public issue tracker! This repository is dedicated to tracking bugs, feature requests, and general feedback for the Steam Update Tracker website.

## About Steam Update Tracker

**Steam Update Tracker** helps you stay informed about game updates you've missed since you last played. Simply sign in with your Steam account, browse your library, and see all the updates for each game in one convenient place.

## Reporting Issues

Found a bug or have a suggestion? We'd love to hear from you! Please follow these guidelines when reporting:

### Before Submitting

- **Search existing issues** to avoid duplicates
- Check if the issue has already been reported or fixed

### How to Report

1. Click on the [Issues](https://github.com/n7z2/steam-update-tracker/issues) tab
2. Click **New Issue**
3. Choose the appropriate template (Bug Report or Feature Request)
4. Fill in all relevant information:
   - **Clear title** describing the issue
   - **Detailed description** of what happened
   - **Steps to reproduce** (for bugs)
   - **Expected behavior** vs actual behavior
   - **Screenshots** if applicable
   - **Browser/Device information**
